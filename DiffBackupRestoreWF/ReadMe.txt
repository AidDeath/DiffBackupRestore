Программа для восстановления дифференциального бэкапа из полной копии BAK и набора TRN файлов
Справка вшита

DiffBackupRestore - первоначальный вариант
DiffBackupRestoreWF - интерфейс переписан под Windows Forms, потому что не у всех установлен VCRedist, а мне лень искать который именно нужен. WF должна работать везде, где есть .Net 3.5

Для обеих версий нужен .Net 3.5
